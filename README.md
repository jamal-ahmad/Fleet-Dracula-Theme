# Fleet-Dracula-Theme

A WIP Dracula theme for Jetbrains Fleet adapted from the default "Fleet Dark" theme

⚠️ terminal ANSI colors need some help!

## Using this theme

`cmd` + `shift` + `K` and then select `Edit Color Theme...` and the paste the entire JSON content of [dark-dracula-edited.json](/dark-dracula-edited.json)
